# 포트폴리오 기획 — 바다 속 포트폴리오

## 콘셉트

> "바다 속에서 수면으로 올라오며 나를 소개한다"

방문자는 깊은 바다 속에서 시작해 수면으로 올라오면서 이가은의 이야기를 만난다.  
깊이 = 시간축 (깊을수록 최신, 얕을수록 과거/마무리)

---

## 깊이별 섹션 구조

| 깊이 | 분위기 | 섹션 |
|---|---|---|
| 가장 깊음 (0m) | 어두운 심해, 거의 검정 | **Intro** — 이름, 타이틀, 입장 |
| 깊음 (−40m) | 짙은 네이비/인디고 | **Latest Work** — 최신 경력/프로젝트 |
| 중간 (−20m) | 중간 블루 | **Projects** — 프로젝트 상세 |
| 얕음 (−10m) | 밝은 하늘빛 | **Skills & Story** — 스킬 + 개발 노트 |
| 거의 수면 (−3m) | 투명한 청록 + 빛 굴절 | **Activities** — 대외활동, 기타 |
| 수면 (0m) | 밝고 환한 하늘 반사 | **Contact** — 이름, GitHub, 이메일 |

---

## 인터랙션 설계

### 이동 방식
- **마우스 휠 / 터치 스와이프** → 깊이 이동 (실제 스크롤 X, depth 값 변경)
- **콘텐츠 클릭** → 해당 섹션으로 smooth 이동
- **Nav 클릭** → 깊이 점프

### 카메라 이동
- Y축 이동 (위 = 수면, 아래 = 심해)
- 이동 중 물속 파티클/먼지 입자 흐름
- 깊이에 따라 **fog density** 변화 (깊을수록 안보임)

### 생물/환경
- **물고기 떼** — 랜덤 경로로 화면 가로지르기 (Boid 알고리즘 or 간단한 sin 경로)
- **파티클** — 수중 먼지/플랑크톤 (위로 천천히 떠오름)
- **빛 굴절** — 수면 근처에서 갓빛(caustics) 효과
- **수면** — 얕은 깊이에서 물결 + 하늘 반사

| 트윈/애니메이션 | GSAP |
| 물리/파티클 | Three.js 직접 구현 |
| UI 오버레이 | React + CSS (3D 위에 HTML 레이어) |
