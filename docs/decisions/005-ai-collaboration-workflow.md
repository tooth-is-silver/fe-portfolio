# 005 — AI와 협업하는 방식

**날짜**: 2026-03-14  
**상태**: 진행중 (프로젝트 전반)

## 배경

이 포트폴리오는 AI 어시스턴트(클로)와 함께 만들고 있다.  
"AI가 코드를 생성해주면 그냥 쓴다"가 아니라 다른 방식을 실험하고 있다.

## 원칙

**나(이가은)가 결정하고, AI가 실행한다.**

- 방향, 기준, 리뷰 → 나
- 구현, 이슈 등록, 브랜치/PR 관리 → AI

## 실제로 협의한 것들

| 주제 | 결정 |
|---|---|
| 브랜치 전략 | 이슈 단위 쪼개기 (가은이 요청) |
| 테스트 범위 | 유틸만 TDD, 컴포넌트 스킵 (같이 논의) |
| 스크린샷 기준 | PC 1280px+ 뷰포트 (가은이 요청) |
| CI 설정 | 테스트 실패 시 머지 블록 (가은이 요청) |
| 콘셉트 전환 | 바다 3D로 전면 전환 (가은이 결정, AI 설계) |
| PR 자동 리뷰 | GitHub Actions + Anthropic API (같이 설계) |

## 느낀 점 (진행중)

AI가 만든 코드라도 내가 리뷰하고 머지를 결정하면 "내 코드"가 된다.  
AI는 실행 속도를 높여주지만 판단은 내가 한다.  
이 경험 자체가 포트폴리오의 일부다.
