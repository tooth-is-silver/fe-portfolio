## 콘셉트 선택 이유

- 바다를 좋아함 — 시원하고 안정감 있는 느낌
- 깊이(depth)가 시간축 역할 → 스크롤이 아닌 깊이 이동으로 내러티브 전달
- 물고기, 파티클, 빛 굴절 → 단순한 포트폴리오가 아닌 경험

## 깊이별 설계

```
심해 (depth 1.0)  → 최신 경력 (어두운 바다)
중간 (depth 0.5)  → 프로젝트
얕음 (depth 0.2)  → 스킬 + 스토리
수면 (depth 0.0)  → 연락처 (밝은 하늘)
```

## 트레이드오프
