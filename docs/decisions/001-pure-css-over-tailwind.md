# 001 — 순수 CSS 선택 (Tailwind/MUI 미사용)

**날짜**: 2026-03-14  
**상태**: 확정

## 고민

포트폴리오를 빠르게 만들려면 Tailwind CSS나 MUI 같은 라이브러리를 쓰는 게 빠르다.  
근데 그러면 결과물이 어디서 본 것 같은 느낌이 된다.

## 결정

순수 CSS + CSS Variables로 직접 작성하기로 했다.

## 이유

- 포트폴리오는 "나를 보여주는 공간" — 남들이랑 비슷하면 의미가 없다
- CSS Variable로 디자인 토큰을 직접 관리 → 의도한 디자인을 정확하게 표현 가능
- 라이브러리 없이도 잘 만들 수 있다는 걸 보여주는 것 자체가 실력 증명

## 트레이드오프

- 더 많은 코드를 직접 작성해야 함
- 반응형 처리를 직접 해야 함 (모바일 퍼스트 min-width 방식으로 해결)

## 이후 변경

3D 바다 콘셉트로 전환하면서 R3F를 도입 → 순수 CSS는 UI 오버레이 부분에만 유지
